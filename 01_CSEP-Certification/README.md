# CSEP Certification Study Hub

This section is organized to mirror the **INCOSE Systems Engineering Handbook v5** chapter structure, which maps directly to the CSEP exam knowledge domains.

## Exam Overview

| Item | Detail |
|------|--------|
| Certifying Body | INCOSE (International Council on Systems Engineering) |
| Credential | Certified Systems Engineering Professional (CSEP) |
| Eligibility | 5+ years SE experience |
| Format | 150 questions, 3 hours |
| Reference | INCOSE SE Handbook v5 + ISO/IEC/IEEE 15288 |
