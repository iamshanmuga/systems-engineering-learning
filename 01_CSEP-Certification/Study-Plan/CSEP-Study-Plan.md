## Key Concepts to Master

- [ ] Systems Thinking and Emergence
- [ ] Life Cycle Model selection criteria
- [ ] All 30 ISO/IEC/IEEE 15288 processes — inputs, outputs, activities
- [ ] Requirements hierarchy (stakeholder → system → subsystem)
- [ ] Architecture Definition vs. Design Definition distinction
- [ ] Verification vs. Validation (V&V)
- [ ] Trade Study methodology (criteria weighting, MOEs, MOPs)
- [ ] Risk management process (identify → analyze → handle → monitor)
- [ ] Technical Performance Measures (TPMs)
- [ ] Configuration Management (CI, baseline types, CCB)
- [ ] Interface Management (ICD, ICDs, N² diagrams)
- [ ] Specialty Engineering integration points
