# Books and References

This folder stores local reference books for CSEP exam preparation and general Systems Engineering study.

> **Note:** PDF and EPUB files in this folder are excluded from the GitHub repository via `.gitignore` because they are large copyrighted materials. Each person should obtain their own licensed copies.

## Books in This Collection (local only)

| File | Description |
|------|-------------|
| `INCOSE Systems Engineering Handbook.epub` | INCOSE SE Handbook — latest version (v5). Primary CSEP exam reference. |
| `INCOSE Systems Engineering Handbook 4e 2015.pdf` | SE Handbook 4th edition (2015) — useful for cross-referencing older process terminology. |
| `INCOSE - Needs and Requirements Manual - Louis S Wheatcraft.pdf` | Deep dive into needs elicitation and requirements management. Key for CH03 topics. |
| `Guide to the Systems Engineering Body of Knowledge v2.10.pdf` | SEBoK v2.10 — broad reference covering the full SE body of knowledge. |
| `Systems Engineering Principles and Practises.pdf` | Foundational SE textbook covering principles and real-world practices. |
| `Essential_Scrum__A_Practical_Guide_to_the_-_Kenneth_S_Rubin.epub` | Essential Scrum by Kenneth Rubin — context for Agile/Scrum within SE life cycle models. |

## Where to Obtain

- **INCOSE publications:** [https://www.incose.org/products-and-publications](https://www.incose.org/products-and-publications)
- **SEBoK:** [https://sebokwiki.org](https://sebokwiki.org) (free online)
- **Other books:** Available via publisher websites or academic libraries.
